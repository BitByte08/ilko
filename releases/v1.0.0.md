# ILKO v1.0.0

Wi-Fi 네트워크에 따라 배경화면을 자동으로 전환하는 macOS 메뉴바 앱입니다.

## 주요 기능

- **자동 전환** — Wi-Fi 게이트웨이 MAC 주소 기반으로 네트워크 변경 감지 → 프로필 자동 적용
- **프로필 관리** — 네트워크별 배경화면 프로필 생성/수정/삭제
- **기본 프로필** — 매칭되는 프로필이 없을 때 자동 적용되는 기본 프로필
- **정적/동영상 지원** — mp4, mov 동영상 및 jpg, png 정적 이미지 모두 지원
- **메뉴바 상주** — 현재 활성 프로필 표시 및 수동 전환 메뉴 제공

## 설치 방법

1. `ILKO-v1.0.0.dmg` 다운로드
2. DMG 열고 `ILKO.app` → Applications 폴더로 드래그
3. 최초 실행 전 Gatekeeper 우회 (Apple 서명 없는 앱):
   ```bash
   xattr -d com.apple.quarantine /Applications/ILKO.app
   ```
4. ILKO 실행 → 메뉴바 아이콘 확인

## 시스템 요구사항

- macOS 14.6 (Sonoma) 이상
